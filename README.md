# Cars Dealership - Full Stack Developer Capstone Project

## Project Overview
A full-stack web application built for **Cars Dealership**, a national car retailer in the U.S.
This app displays dealership branches across the country and allows users to view and submit reviews.

## Technologies Used
- **Frontend:** React, HTML, CSS
- **Backend:** Django, Node.js, Flask
- **Database:** SQLite, MongoDB
- **Deployment:** Docker, Kubernetes, IBM Cloud Code Engine
- **CI/CD:** GitHub Actions
- **Sentiment Analysis:** IBM Watson / Custom Flask service
